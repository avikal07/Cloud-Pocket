# Cloud-Pocket ☁️💰

Cloud-Pocket is a serverless expense tracking application that helps users manage and monitor their expenses using cloud services.

## Features
- Track daily expenses
- Serverless backend
- Cloud storage for data
- Simple and responsive interface

## Tech Stack
- Node.js
- Serverless framework
- Cloud services (AWS / Firebase etc.)
- JavaScript

│
├── backend
